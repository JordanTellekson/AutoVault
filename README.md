# AutoVault

**Final Project for Mobile App Development**

AutoVault is a mobile-focused Blazor Hybrid application for managing a personal car collection. It allows users to add, edit, and delete cars, track maintenance records, and view detailed specifications for each vehicle.

---

## Features

### Car Collection Management
- Add new cars with full specifications
- Edit or delete existing cars
- View cars in a responsive grid layout

### Maintenance Tracking
- Add, edit, and delete maintenance records
- Automatic sorting of records from oldest to newest
- Inline editing and deletion via modal confirmation

### Detailed Vehicle Pages
- Engine, performance, and miscellaneous specs
- Image display (supports both local paths and external URLs)
- Carbon fiber flag and other detailed features

### Responsive Design
- Optimized for mobile, tablet, and desktop
- Grid layout adapts to screen width
- Cards maintain proper spacing and clickable navigation

### Data Persistence
- Uses local JSON file storage (`cars.json`)
- All CRUD operations persist to JSON
- Supports caching for performance
